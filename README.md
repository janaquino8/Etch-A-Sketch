# Etch-A-Sketch
A simple web replication of the Etch A Sketch drawing toy

## Features
- Generation of boards of up to 100 x 100 px
- Varying color options

## Cloning Guide
```sh
git clone https://github.com/janaquino8/Etch-A-Sketch.git
```

*Inspired from [The Odin Project](https://www.theodinproject.com/lessons/foundations-etch-a-sketch)*
